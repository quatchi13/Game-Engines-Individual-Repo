Controls:

Arrow Keys to move
Left Click to Shoot
ESC key to enter edit mode (game scene only)

Game Info:
Shoot the Goblin off his platform to win. pick up coins for extra points 
	and hearts for extra lives. If you run out o flives it's game over.

References:
Parisa's Code Base used to expand off of
Provided models from lab found here: https://quaternius.com/packs/ultimateplatformer.html
TextMeshPro library used for UI text
Unity Documentation for general use/debugging
